# NÉXUS • DARK MOON

Telegram Mini App — Pokémon-подобная игра с элементами Смешариков и Dark Moon.

### Что уже готово
- Полноценный overworld на Phaser 3 (бесконечная карта чанками)
- Боевая система с HP-барами, логами и сменой покемонов
- Lunadex с поиском и фильтрами
- Команда, инвентарь и система сохранений
- Все данные (покемоны + Смешарики + Dark Moon)

### Как запустить
Открой `index.html` через Live Server или закинь репозиторий на GitHub Pages.
